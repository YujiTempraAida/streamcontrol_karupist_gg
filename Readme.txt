Smash.gg関連機能について

経緯：
対戦組み合わせを毎回Smash.ggからコピペするのが面倒。
Smash.ggが提供するAPIを利用して、スコアボードの自動更新ができればオペレータの作業が減り、よりクリエイティブなタスクへリソースを割り当てられるようにする狙い。

「ggをインポート」にチェックを入れて保存すると、
- prefix（スポンサー名）
- プレイヤー名
- ラウンド
- スコア
を一番直近の配信台カードの情報を自動で読み取って更新します。

チェックを外して保存すると通常の手入力StreamControlに戻ります。

トークン作成はこちら参照
https://developer.smash.gg/docs/authentication/
